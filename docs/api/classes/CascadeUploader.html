    Class CascadeUploader

    CascadeUploader manages the complete file upload workflow.

    This class integrates all components of the storage layer to provide a simple, high-level API for uploading files to Cascade. It handles:

    • File hashing and integrity verification
    • RaptorQ layout generation
    • LEP-1 compliant index file construction
    • Upload task initiation and monitoring

    Remarks

    The current implementation uses a simulated signature for start_signature. In production, this should be replaced with a real wallet signature over the BLAKE3 hash of the file data using ADR-036 signArbitrary.

    • uploadFile(
          file: ArrayBuffer | Uint8Array<ArrayBufferLike> | Blob,
          params: UploadParams,
      ): Promise<{ progress?: number; status?: string; taskId?: string }>

      Upload a file to Cascade storage

      This method orchestrates the complete upload workflow:

      1. Converts the file to bytes and calculates data_hash (BLAKE3)
      2. Generates a single-block RaptorQ layout using rq-wasm
      3. Derives 50 layout IDs per LEP-1 specification
      4. Constructs the index file with layout metadata
      5. Simulates the start_signature (placeholder for wallet integration)
      6. Initiates upload via sn-api with multipart form data
      7. Monitors the upload task until completion
      8. Returns the completed task details

      Parameters

      • file: ArrayBuffer | Uint8Array<ArrayBufferLike> | Blob

        The file to upload (as Blob, ArrayBuffer, or Uint8Array)

      • params: UploadParams

        Upload parameters including action ID and blockchain params

      Returns Promise<{ progress?: number; status?: string; taskId?: string }>

      Promise resolving to the completed upload task

      Throws

      If any step of the upload workflow fails

      Example

      const uploader = new CascadeUploader(snapiClient);

      // Upload from a file input
      const fileInput = document.querySelector('input[type="file"]');
      const file = fileInput.files[0];

      const result = await uploader.uploadFile(file, {
        actionId: 'action-123',
        rq_ids_ic: 1000,
        rq_ids_max: 10000,
        taskOptions: {
          pollInterval: 2000,
          timeout: 300000
        }
      });

      console.log('Upload complete:', result);
