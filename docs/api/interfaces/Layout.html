    Interface Layout

    RaptorQ layout generated by rq-wasm for a single block.

    This layout contains the parameters needed for RaptorQ encoding/decoding, including the transfer length, symbol size, and block structure.

    Remarks

    The layout is deterministic based on the input file bytes and is used to derive layout IDs according to LEP-1 specifications.
